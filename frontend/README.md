# IV_CODE

iv_Code is a website template for fetching covid examinations data from backend.

## Installation

Use the package manager [yarn](https://yarnpkg.com) or [npm](https://npmjs.com) to install packages.

```bash
yarn install
```

or

```bash
npm install
```

## Usage

run project in development mode

```bash
yarn dev
```

or

```bash
npm run dev
```
